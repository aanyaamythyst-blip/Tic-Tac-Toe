# ❌⭕ Tic Tac Toe Game

A simple **desktop Tic Tac Toe game** built with Python.
This application allows two players to play Tic Tac Toe while tracking scores across multiple rounds.

It’s a beginner-friendly project created to practice **Python GUI development, game logic, and user interaction.**

---

## ✨ Features

* Two-player **Tic Tac Toe gameplay**
* Players enter their **names before the game starts**
* Choose **best-of number of rounds**
* Automatic **win detection**
* **Score tracking** for both players
* Option to **reset the board anytime**
* Simple **graphical interface**

---

## 🧠 How It Works

1. When the program starts, players enter their **names**.
2. Players choose **how many rounds** they want to play (best-of format).
3. The game displays a **3×3 Tic Tac Toe board**.
4. Players take turns placing **X and O**.
5. The program automatically checks for **winning combinations**.
6. The score updates when a player wins a round.

---

## 🛠️ Technologies Used

This project uses the following technologies:

* **Python** – Main programming language
* **Tkinter** – Used to build the graphical user interface (GUI)
* **Tkinter MessageBox** – Displays game notifications and results
* **Tkinter SimpleDialog** – Used to collect player names and game settings

## 🚀 How to Run the Project

1. Make sure **Python 3** is installed on your computer.

2. Clone this repository:

```bash
git clone https://github.com/yourusername/tic-tac-toe-python.git
```

3. Navigate to the project folder:

```bash
cd tic-tac-toe-python
```

4. Run the program:

```bash
python tictactoe.py
```
